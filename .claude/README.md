## Быстрый старт

```bash
cd ~/Desktop/project
claude
```

Промпт — **intent-based**: что делаем, как должно работать для игрока, как проверить. Не список файлов.

```
/pqs Phase 1: игрок получает базу и видит монеты
```

---

## Команды

| Команда | Когда |
|---------|-------|
| `/pqs` | Сформулировать промпт (не выполнять) |
| `/context` → `/compact` @ 65% | Длинная сессия |
| `/clear` | Застряли |
| `/effort` | low = поиск, high = код |
